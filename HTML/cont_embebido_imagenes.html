Imagenes

Las imagenes web se pueden clasificar en 2 tipos

  • Vectoriales

    svg recomendado siempre que se pueda

  • Mapa de bits

    jpg

    png 8 y 24 si necesitas transparencia

    git si necesitas una imagen animada

    webp el formato que menos pesa, se usan para imagenes que no pueden ser vertorizadas y su motor de render es la misma web

Insertar imagen

Se inserta con la etiqueta img y tiene dos atributos src="" que es la ruta donde se encuentra la imagen y alt="" es un texto alternativo que me mostrar√° si falla la subida de la imagen falla, la etiqueta imagen soporta todas las extensiones de imagenes (png, jpg, gif, webp, jpeg)

Tambien hay dos etiquetas que se le pueden colocar with y height Los cuales son para determinar el "ancho" y el "alto" respectivamente pero el ancho es estatico entonces cuando cambiamos el AspectRadio, para eso podemos mediante CSS agregarle la propiedad max.width: n%

Las imagenes svg tienen su propia etiqueta y tiene diferntes atributos que se pueden modificar en CSS.

srcset

Es un atributo que nos permite cargar una imagen u otra dependiendo del devicePixelRatio, se pasan varios "src" separados por comas y al frente se coloca nx dependiendo del ApectPixelRadio, podemos optimizar el codigo o tener "una copia de seguraridad", se tiene un srcset y si este no sirve o no carga, se puede colcoar un src normal para que muestre algo de contigencia

picture

trabaja igual que srcset solo que la imagen por default va dentro con un src y se puede establecer con media="(max-width:500px)" para que despues de ese ancho, se carge otra imagen
