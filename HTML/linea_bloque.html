Diferencia de elementos de bloque y elementos de linea

Elementos de bloques

Son aquellos que ocupan todo el largo de la pagina

address

Se utiliza para aportar información de contacto para el article más cercano o para todo el body

Tony Stark
Barranquilla
080001

blockquote

Se utiliza para citar otros autores, se puede incluir el atributo "cite" para hacer un enlace al documento original o fuente

Aquellos que estan dispuestos a disparar tambien deben estar dispuesto a ser disparados

pre

Se utiliza para tener codigo preformateado que necesita ser representado igual que se escribio

Esto      esta escrito         tal cual 
                lo escribí en                       el              html

div

Se itiliza como division del documento, semanticamente significa nada, es un contenedor generico que se usa generalmente para dar estilos a traves de css o para usar algo denominado "delegación de eventos" en javascript

Nombre

email

descripción

hr

significa "horizontal rule" y se utiliza para decirle al navegador que vamos a cambiar de tema

Elementos de linea

Son aquellos cuyo contendor es del largo del contenido y pueden darle algunos estilos a las palabras que las contienen, estas son : em , strong , small , br , wbr , time , i , b , u, sup, sub .

Es del ancho de su contenido

span

contenedor de linea que sirve para darle estilos con CSS o identificarlos con JS

q

equivalente a blocquote pero en linea Yo soy IronMan

code

Sirve para encerrar codigo que queremos representar visualmente, suele ir unido con la etiqueta pre 
                
                    import cv2
                    video = cv2.VideoCapture(0)
                    while True:
                        ret,frame = video.read()
                        if ret == False:
                            break

                        cv2.imshow('video',frame)

                        q = cv2.waitKey(1)
                        if q == ord('q'):
                            break
                    cv2.destroyAllWindows()
                    video.release()
