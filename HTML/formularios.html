Tips

  • cada label junto con su respectivo input, lo podemos meter en un mismo "div"
  • Lo que hace el atributo "name" es que agrupa campos, lo que hace que solo se pueda marcar uno de ellos
  • La etiqueta "value" lo que hace es darle un valor a el campo, de modo que si se selecciona ese campo y se envia, se enviara dicho valor
  • La etiqueta "checked" sirve para que el campo venga marcado por default

Se utiliza para agrupar elementos dentro de un formulario

Se utiliza para colocarle nombre al campo creado por el fieldset

Representa un valor dentro de un rango conocido, recibe como atributos "min", "max" y value que es el valor a mostrar, "low" para decir que lo que este debajo de ese valor, estará en rojo, "high" para que lo que esté encima de ese valor este en amarillo
progress
Representa el progreso de una tarea, parecido al meter
textarea
Se utiliza para introducir texto en un formulario
